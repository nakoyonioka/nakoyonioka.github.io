Technical Deep Dive

Project Overview

This adjustable power supply delivers voltage regulation with protection circuitry. Featuring dual output capabilities (selectable 5V fixed or 0-12V adjustable) and adjustable current limiting, it is a versatile tool for electronics work.

Technical Specifications

Input Voltage 12V DC
Output Options Selectable 5V fixed / 0-12V DC adjustable
Max Current Up to 3A (adjustable limit)

Technical Architecture

Circuit Design Flow

Input & Filtering Stage

12V DC input with input fuse (F301, 5A) protection and multi-capacitor filtering.

Multi-Cap Filtering Terminal Block Input Connector (e.g., J301, J302)

Dual Regulation

Combines linear regulation (L7805 type, U301, for 5V output) and switching regulation (TPS54331DR, U302, for adjustable 0-12V output).

L7805 Type Linear (U301) TPS54331DR Switch (U302) Output Voltage Potentiometer (RV301)

Output Selection

Relay-based switching (K401) selects between the 5V fixed and the adjustable 0-12V outputs, routed to banana plug connectors (J402, J403).

Relay Switching (K401) Output Isolation via Relay

Current Protection & Output Control

Adjustable current limiting using a comparator (LM393 type, U401A) and sense resistor (R403, 100mΩ). Features hysteresis (R401, R402 with U401A) to prevent oscillation. Overcurrent condition can trigger output shut-off via relay K403.

LM393 Comparator (U401A) 100mΩ Sense Resistor (R403) Adjustable Threshold (RV401) Hysteresis Circuit Output Shut-off Relay (K403)

Measurement & Output

Provides connections for external voltmeter (J404) and current meter (J406). Includes an LM358 op-amp (U402A) based circuit for internal current measurement (DMEASCUR signal). Output via banana plugs (J402, J403).

Op-Amp Current Measurement (U402A) Banana Plug Outputs (J402, J403) External Meter Connections (J404, J406)

Circuit Design Principles

Linear regulator (L7805 type, U301) for the 5V output, offering stable voltage
Switching regulator (TPS54331DR, U302) for the efficient, adjustable 0-12V output
Dedicated feedback networks for voltage regulation (e.g., R305, R306, RV301 for U302)
Comparators (LM393 type, U401) for current limit detection

PCB Layout Strategies

Four-layer PCB with dedicated power and ground planes
Wide copper traces intended for main power paths
Thermal relief and copper pours considered for heat dissipation

Protection Systems

Main input fuse (F301, 5A) for overcurrent protection
Current limiting with a 100mΩ sense resistor (R403)
Adjustable current threshold (via RV401) up to 3A
Hysteresis circuit (e.g., R401, R402 with U401A) to prevent current limit oscillation
Overcurrent detection can trigger output shut-off using relay K403

Measurement System

Differential amplifier configuration (U402A, LM358) for current measurement across R403
Low-pass filtering (e.g., C407 on VMETER, C410 on DMEASCUR) on measurement signal paths
Connection points (J404, J406) for external voltage and current meters
