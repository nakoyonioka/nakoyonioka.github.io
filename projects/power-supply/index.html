Project Overview

This adjustable power supply delivers precision voltage regulation with advanced protection circuitry for professional electronics development. Featuring dual output capabilities and intelligent current limiting, it represents an essential tool for any electronics laboratory.

Technical Specifications

Input Voltage 18V - 24V DC
Output Options 5V / 12V DC
Max Current 3A (adjustable)
Ripple Voltage < 50mV
Load Regulation < 1%

Technical Architecture

Circuit Design Flow

Input & Filtering Stage

18V-24V DC input with comprehensive transient protection and multi-stage filtering

TVS Protection Multi-Cap Filtering Barrel Jack Input

Dual Regulation

Hybrid linear and switching regulation for optimal efficiency and noise performance

L7805 Linear TPS54331DR Switch Thermal Protection

Output Selection

Relay-based switching between 5V and 12V outputs with complete isolation

Relay Switching LED Indicators Isolation

Current Protection

Fast-response current limiting with precision sensing and hysteresis control

LM393 Comparator 100mΩ Sense <10μs Response

Measurement & Output

Precision measurement circuits with banana plug connectors for laboratory use

Differential Amp Banana Plugs Ground Reference

Circuit Design Principles

Linear regulation provides clean 5V with excellent noise characteristics
Switching regulation delivers efficient 12V with 85% efficiency
Dedicated feedback networks ensure tight regulation
Fast-response comparators for rapid fault detection

PCB Layout Strategies

Four-layer PCB with dedicated power and ground planes
Wide copper traces (>60 mil) for main power paths
Star grounding topology for analog sections
Thermal relief patterns and copper pour for heat dissipation

Protection Systems

Precision current limiting with 100mΩ sense resistor
Adjustable current threshold from 100mA to 3A
Hysteresis circuit prevents oscillation at threshold
TVS diodes for input transient protection

Measurement System

Differential amplifier using LM358 for current measurement
Precision 1% resistors for voltage measurement
Low-pass filtering on measurement paths reduces noise
Connection points for external meters
