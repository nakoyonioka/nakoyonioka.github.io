Module 4: Active Filters

Low-Pass and High-Pass Design

Filters are essential circuits that selectively pass or block signals based on their frequency. While passive filters (using only resistors, capacitors, and inductors) exist, active filters incorporate active components like op-amps. This allows them to achieve higher gain, better performance, and avoid loading effects, all without using bulky inductors.

In this final module of the workshop, you'll design and build a first-order low-pass filter and understand the principles of a high-pass filter.

What You'll Need:

  • A breadboard
  • A dual DC power supply (e.g., +/- 9V)
  • One LM741 Op-Amp IC (or TL071/TL081)
  • Resistors:
    • Two 10kΩ
    • One 1kΩ (for gain setting)
  • Capacitors:
    • One 0.1µF ceramic capacitor
  • Jumper wires
  • A function generator (recommended for testing frequencies)
  • An oscilloscope (essential for visualizing filter response)
  • (Optional: An audio source and speaker/headphones for qualitative testing)

Circuit Theory: Filter Types

  • Low-Pass Filter (LPF): Allows frequencies below a certain cutoff frequency ($f_c$) to pass through and attenuates frequencies above it. Useful for removing high-frequency noise.
  • High-Pass Filter (HPF): Allows frequencies above a certain cutoff frequency ($f_c$) to pass through and attenuates frequencies below it. Useful for removing low-frequency hum or DC offset.

The cutoff frequency for a first-order RC (Resistor-Capacitor) filter is:

$$f_c = \frac{1}{2\pi RC}$$

Circuit: First-Order Active Low-Pass Filter

We'll build a non-inverting active low-pass filter. This design provides both filtering and amplification.

Circuit Diagram (Active Low-Pass Filter):

  1. Connect Pin 7 (V+) and Pin 4 (V-) of the op-amp to your positive and negative power supplies.
  2. Input Stage (RC Filter): Connect one 10kΩ resistor ($R$) from your signal input to Pin 3 (Non-Inverting Input) of the op-amp.
  3. Connect the 0.1µF capacitor ($C$) from Pin 3 (Non-Inverting Input) to GND.
  4. Feedback Loop (Amplifier): Connect the other 10kΩ resistor ($R_f$) from Pin 6 (Output) to Pin 2 (Inverting Input).
  5. Connect the 1kΩ resistor ($R_i$) from Pin 2 (Inverting Input) to GND.
  6. The output is taken from Pin 6.

For $R = 10k\Omega$ and $C = 0.1\mu F$, the cutoff frequency is: $$f_c = \frac{1}{2\pi \times 10000 \Omega \times 0.0000001 F} \approx 159 Hz$$ The gain of the amplifier stage is $1 + \frac{R_f}{R_i} = 1 + \frac{10k\Omega}{1k\Omega} = 11$.

Testing the Filter:

This is where a function generator and oscilloscope are invaluable.

  1. Connect your function generator to the input of the filter.
  2. Connect the oscilloscope probes to the input and output of the filter.
  3. Set the function generator to output a sine wave with a small amplitude (e.g., 0.5V peak-to-peak).
  4. Below Cutoff Frequency (e.g., 50 Hz): Observe the input and output on the oscilloscope. The output amplitude should be about 11 times the input amplitude (due to the gain).
  5. At Cutoff Frequency (e.g., 159 Hz): The output amplitude should be approximately $0.707 \times (Gain \times Input\_Amplitude)$ (i.e., about 70.7% of the maximum amplified signal).
  6. Above Cutoff Frequency (e.g., 1 kHz): As you increase the frequency, you'll see the output amplitude significantly decrease compared to the input, demonstrating the filtering action.

If you don't have a function generator/oscilloscope, you can try playing audio with varying frequencies (e.g., bass-heavy music vs. treble-heavy music) and listen to the output. The low-pass filter should make bass sounds more prominent and high-pitched sounds quieter.

High-Pass Filter Concept:

An active high-pass filter can be created by simply swapping the positions of the resistor and capacitor in the RC input stage of the low-pass filter. The capacitor would be in series with the input, and the resistor would go from the non-inverting input to ground. The gain stage remains the same.

Troubleshooting Tips:

  • No filtering effect?
    • Component values: Double-check your resistor and capacitor values. Even small errors can shift the cutoff frequency.
    • Connections: Ensure the RC components are correctly placed at the non-inverting input and the feedback resistors are correct.
    • Power supply: Verify the op-amp is powered correctly.
  • Gain issues? Revisit the op-amp gain calculation and component values for the feedback loop.
  • Testing setup: Ensure your function generator is outputting a clear signal and your oscilloscope is correctly configured to display the waveforms.

You've successfully built and tested an active filter! Filters are ubiquitous in electronics, from audio equalizers to sensor data conditioning and communication systems. This module provides a solid foundation for understanding how to shape signals based on frequency.
