Module 2: Button Input

Reading Digital Signals

In the previous module, you learned how to make an LED blink by controlling a digital output pin. Now, let's make your Arduino project interactive by reading input from a button! This is a fundamental concept for creating user interfaces and responding to external events.

You'll learn about digital input pins, pull-up/pull-down resistors, and how to write code to detect when a button is pressed.

What You'll Need:

  • An Arduino board
  • A USB cable
  • A breadboard
  • One pushbutton (4-pin tactile switch)
  • One 10kΩ resistor (for pull-down or pull-up)
  • One LED
  • One 220 Ohm resistor (for LED)
  • Jumper wires

Circuit Diagram (Pull-Down Resistor):

We'll use a pull-down resistor configuration. This means when the button is NOT pressed, the input pin is pulled LOW (0V). When the button IS pressed, the input pin is connected to HIGH (5V).

  1. Connect one leg of the pushbutton to Arduino 5V.
  2. Connect the diagonally opposite leg of the pushbutton to Arduino Digital Pin 2.
  3. From the same leg of the pushbutton connected to Pin 2, add the 10kΩ resistor and connect its other end to GND. This is your pull-down resistor.
  4. Connect the LED circuit as in Module 1: Arduino Digital Pin 13 -> 220 Ohm resistor -> LED anode (long leg) -> LED cathode (short leg) -> GND.
  5. Ensure all GND connections are tied together to the Arduino GND.

Arduino Code (Sketch):

This sketch will read the state of the button and turn the LED on when the button is pressed, and off when it's released. 


const int buttonPin = 2;     // the number of the pushbutton pin
const int ledPin = 13;       // the number of the LED pin

// variable for storing the pushbutton state
int buttonState = 0;

void setup() {
  // initialize the LED pin as an output:
  pinMode(ledPin, OUTPUT);
  // initialize the pushbutton pin as an input:
  pinMode(buttonPin, INPUT);
}

void loop() {
  // read the state of the pushbutton value:
  buttonState = digitalRead(buttonPin);

  // check if the pushbutton is pressed.
  // if it is, the buttonState is HIGH:
  if (buttonState == HIGH) {
    // turn LED on:
    digitalWrite(ledPin, HIGH);
  } else {
    // turn LED off:
    digitalWrite(ledPin, LOW);
  }
}

Key parts of this code:

  • const int buttonPin = 2;: Defines a constant for the button pin (Pin 2).
  • pinMode(buttonPin, INPUT);: Configures Pin 2 as an input.
  • digitalRead(buttonPin);: Reads the digital value (HIGH or LOW) from the button pin.
  • if (buttonState == HIGH) { ... } else { ... }: An if-else statement checks the `buttonState`. If HIGH (button pressed), the LED turns on. Otherwise (button not pressed), it turns off.

Uploading and Testing:

  1. Connect your Arduino board to your computer.
  2. Select the correct board and port in the Arduino IDE (Tools > Board, Tools > Port).
  3. Copy and paste the code into your Arduino IDE, then click the Upload button.

Press the button! The LED should light up when you press it and turn off when you release it.

Using Internal Pull-Up Resistors (Alternative):

Arduino microcontrollers have internal pull-up resistors that you can enable in code, eliminating the need for an external resistor. If you use an internal pull-up, the button should be wired to connect the input pin to GND when pressed.

To use the internal pull-up:

  1. Remove the 10kΩ external pull-down resistor from your circuit.
  2. Connect one leg of the pushbutton to Arduino Digital Pin 2.
  3. Connect the diagonally opposite leg of the pushbutton to GND.

Modify the setup() function in your code:


void setup() {
  pinMode(ledPin, OUTPUT);
  // initialize the pushbutton pin as an input with internal pull-up:
  pinMode(buttonPin, INPUT_PULLUP);
}

And in the loop(), since the button will read LOW when pressed (pulled to GND), reverse the logic: 


void loop() {
  buttonState = digitalRead(buttonPin);

  if (buttonState == LOW) { // Button pressed, pin is LOW
    digitalWrite(ledPin, HIGH);
  } else { // Button not pressed, pin is HIGH due to internal pull-up
    digitalWrite(ledPin, LOW);
  }
}

Upload this modified code and test again. Both methods achieve the same result, but internal pull-ups save external components.

Troubleshooting Tips:

  • LED always on or off, regardless of button:
    • Double-check your wiring, especially the resistor connection for the button.
    • Ensure the button is correctly inserted and making contact.
    • If using internal pull-up, ensure your button connects the pin to GND when pressed.
  • "Floating" input (unpredictable behavior): This happens if you don't use a pull-up or pull-down resistor. The pin's state will be random. Always use one!

You've successfully integrated a button into your Arduino project! This opens up a world of possibilities for user interaction. In the next module, you'll learn how to read analog inputs, which are essential for interfacing with many types of sensors.
