Module 1: Schematic Capture

Designing Your Circuit Diagram in KiCad

Schematic capture is the very first step in the PCB design process. It's where you translate your circuit idea into a formal, graphical representation using standard electronic symbols. A well-designed schematic is crucial for clarity, debugging, and ensuring your PCB functions correctly.

In this module, you'll learn the basics of using KiCad's schematic editor (Eeschema) to place components, draw wires, and organize your circuit diagram.

What You'll Need:

  • KiCad installed on your computer
  • A basic understanding of electronic components (resistors, capacitors, LEDs, etc.)

KiCad Project Setup:

Before you start drawing, create a new KiCad project:

  1. Open KiCad.
  2. Go to File > New Project > New Project...
  3. Choose a location and name your project (e.g., "MyFirstPCB").
  4. KiCad will create a project folder containing a `.pro` file, a `.sch` file (for schematic), and a `.kicad_pcb` file (for PCB layout).
  5. Double-click the `.sch` file (or click the "Schematic Editor" icon in the KiCad Project Manager) to open Eeschema.

Placing Components:

Components are the building blocks of your schematic. KiCad has a vast library of symbols.

  1. Press 'A' (for "Add Symbol") or click the "Place symbol" icon () on the right toolbar.
  2. In the "Choose Symbol" dialog, type to search for components. For example:
    • "R" for resistor
    • "C" for capacitor
    • "LED" for Light Emitting Diode
    • "GND" for ground symbol
    • "VCC" or "5V" for power symbols
  3. Select a component and click OK. Place it on the schematic sheet.
  4. Repeat to add all necessary components for a simple blinking LED circuit:
    • 1 x Arduino Uno (search for "Arduino_Uno_R3")
    • 1 x LED (search for "LED")
    • 1 x Resistor (search for "R")
    • 1 x Power Flag (search for "PWR_FLAG")
    • 1 x Ground (search for "GND")

Wiring Components:

Once components are placed, you need to connect them with wires.

  1. Press 'W' (for "Wire") or click the "Place wire" icon () on the right toolbar.
  2. Click on the end of one component's pin and drag the wire to the pin of another component. Click again to place a corner or finish the wire.
  3. Connect the LED and resistor in series, then connect this series to an Arduino digital pin (e.g., D13) and GND.
  4. Connect the 5V output of the Arduino to a "PWR_FLAG" symbol, and the GND of the Arduino to a "GND" symbol. These power symbols are important for KiCad's Electrical Rules Check (ERC).

Annotating and Numbering:

Each component needs a unique reference designator (e.g., R1, C1, U1).

  1. Click the "Annotate schematic symbols" icon () on the top toolbar.
  2. Click "Annotate" in the dialog box. KiCad will automatically assign unique numbers (e.g., R1, R2, D1).

Electrical Rules Check (ERC):

ERC checks for common errors in your schematic, like unconnected pins or power conflicts.

  1. Click the "Perform electrical rules check" icon () on the top toolbar.
  2. Click "Run ERC".
  3. Review any warnings or errors. Fix them before proceeding!

Netlist Generation:

The netlist is a text file that describes all component connections. It's used to transfer the schematic information to the PCB layout tool.

  1. Click the "Generate netlist" icon () on the top toolbar.
  2. Ensure "Pcbnew" tab is selected.
  3. Click "Generate Netlist". Save it in your project folder.

You've successfully completed schematic capture for a basic circuit in KiCad! This is the blueprint for your PCB. A clean and error-free schematic is paramount for a smooth PCB layout process.
