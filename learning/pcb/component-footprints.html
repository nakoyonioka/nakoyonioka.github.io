Module 2: Component Footprints

Linking Schematic Symbols to Physical Parts

After creating your schematic, the next crucial step in PCB design is associating each schematic symbol with its corresponding footprint. A footprint (also known as a land pattern or package) is the physical representation of a component on the PCB. It defines the size, shape, and arrangement of the pads where the component will be soldered.

In this module, you'll learn how to assign footprints to the symbols in your KiCad schematic and understand the importance of choosing the correct footprint.

What You'll Need:

  • Your completed schematic from Module 1 (e.g., the blinking LED circuit)
  • KiCad installed

Understanding Footprints:

A single component, like a resistor, can come in many different physical packages (sizes). For example:

  • Through-hole (THT) resistors: Axial leads, common sizes like 0.25W, 0.5W.
  • Surface-mount (SMD) resistors: Tiny rectangular packages like 0805, 0603, 0402.

Each of these physical packages requires a different footprint on the PCB. Choosing the correct footprint is vital for manufacturing and assembly.

Assigning Footprints in KiCad:

KiCad's "Assign Footprints" tool (CvPcb in older versions) helps you link schematic symbols to footprints.

  1. From the KiCad Project Manager, click the "Assign footprints to schematic symbols" icon (). This will open the Footprint Assignment Tool.
  2. The window is typically divided into three columns:
    • Left: Available footprint libraries.
    • Middle: Symbols from your schematic (e.g., R1, D1, U1).
    • Right: Footprints available in the selected library.
  3. For each symbol in the middle column:
    • Select the symbol (e.g., "R1").
    • In the left column, select a relevant library (e.g., "Resistor_SMD" for SMD resistors, or "Resistor_THT" for through-hole).
    • In the right column, choose the appropriate footprint for your component. For a standard 0805 SMD resistor, select "Resistor_SMD:R_0805_2012Metric". For a through-hole resistor, select "Resistor_THT:R_Axial_DIN0207_L6.3mm_D2.5mm_P7.62mm".
    • Double-click the desired footprint to assign it to the selected symbol. The assigned footprint will appear next to the symbol in the middle column.
  4. Assign footprints for all components:
    • LED: Search for "LED_SMD" or "LED_THT" and choose a suitable size (e.g., "LED_SMD:LED_0805_2012Metric").
    • Arduino Uno: This is a bit special. For simplicity, you might use a header footprint for the pins, or if you plan to solder headers to a custom board, search for "Connector_PinHeader_2.54mm" and select appropriate sizes (e.g., "PinHeader_1x08", "PinHeader_1x06"). For a full Arduino board, you might need to create a custom footprint or use a pre-existing one if available in your KiCad version. For this exercise, let's assume you're connecting wires to the Arduino pins, so you don't necessarily need a full board footprint on *your* custom PCB, but rather header pins.
  5. Once all footprints are assigned, click "Apply, Save Schematic & Continue" or "Apply, Save Schematic & Exit".

Footprint Editor (Optional - Advanced):

Sometimes, you might need a custom footprint that isn't in the standard libraries. KiCad's Footprint Editor allows you to create your own.

  1. From the KiCad Project Manager, click the "Footprint Editor" icon ().
  2. You can create new footprints from scratch or modify existing ones. This involves drawing pads, silkscreen outlines, and defining 3D models.

You've successfully assigned footprints to your schematic symbols! This step bridges the gap between the logical circuit diagram and the physical layout of your PCB. With footprints assigned, you're ready to move on to the actual board layout.
